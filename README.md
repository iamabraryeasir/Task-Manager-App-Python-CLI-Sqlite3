# Task Manager App (Python CLI) 📝🐍

## Key Features:

-   📋 Listing the tasks.
-   ➕ Adding new task.
-   ✏️ Updating an existing task.
-   🗑️ Deleting a task.
-   📊 Dynamic table for task showcase.
-   💾 Saving the tasks on an SQlite 3 Data Base.
